게시판을 만든다.

1. Spring, Spring MVC, Spring JDBC
2. Session 을 사용하여 로그인처리
3. MySQL
4. 주요기능 : 글목록보기, 저장하기, 수정하기, 삭제하기, 로그인, 로그아웃
댓글달기, 페이징처리, 파일업로드, 파일다운로드

글쓰기,글수정,글삭제,댓글달기는 로그인 후에 사용가능.
id, password를 이용하여 로그인을 한다.

게시판은 카테고리를 가지고 있다.

0) 우리가 만들 게시판에 대한 UI 프로토타이핑을 한다.
   어떤 URL이 필요한지 도출한다.
   각각의 URL에서 어떤 비지니스 로직이 필요한지 도출한다.
1) 테이블 생성 SQL을 작성. sample data를 insert한다.
(페이징 처리와 관련된 sql을 확인을 위해 실제 같은 데이터를 저장한다.)
2) 필요한 SQL이 무엇일까? SQL부터 작성을 한다. (조별로 검사를 받는다.)
3) JDBC 설명, Spring JDBC에 대하여 설명
4) Spring JDBC를 이용해서 Repository를 작성
5) Repository를 테스트 한다. (조별로 검사를 받는다.)
6) Service 객체를 생성한다. 테스트한다. (조별로 검사를 받는다.)
7) Controller객체를 생성한다. 브라우저로 확인
